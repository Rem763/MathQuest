# 数字闯关技术说明

**项目名称**：数字闯关（Number Quest）  
**项目形态**：JavaFX 单机桌面游戏  
**技术栈**：Java 17、JavaFX 21、Maven、本地 JSON 存档  
**适用平台**：Windows 10/11

---

## 1. 项目概述

### 1.1 项目简介

数字闯关是一款面向小学生的像素风数学闯关游戏。项目将基础算术题与回合制战斗机制结合，玩家通过答题推进关卡，在地图、战斗、结算三个核心场景之间完成完整流程。

### 1.2 设计目标

- 以答题驱动闯关，提升学习趣味性
- 使用 JavaFX 实现轻量、直观的桌面交互
- 采用本地 JSON 存档，保证离线可用
- 使用统一资源命名与场景结构，提高维护性

### 1.3 系统特点

- 单机运行，不依赖后端服务
- 像素风界面与角色动画
- 地图选关与难度切换
- 回合制答题战斗
- 胜负结算与继续游戏
- 本地 JSON 自动存档

---

## 2. 功能概览

### 2.1 主菜单

- 新游戏
- 继续游戏

### 2.2 地图系统

- 显示当前关卡与玩家状态
- 普通 / 困难难度切换
- 关卡入口与 BOSS 关入口
- 返回标题页

### 2.3 战斗系统

- 四选一答题
- 整数加减乘除题目
- 普通模式 10 秒倒计时
- 困难模式 3 秒倒计时
- 正确 / 错误 / 超时反馈
- 玩家与怪物逐帧动画
- 逃跑按钮
- 战斗结束结算

### 2.4 结算系统

- 胜利结算
- 失败结算
- 星级与金币奖励
- 下一关、复活、返回地图

### 2.5 存档系统

- 本地 JSON 自动保存
- 继续游戏恢复进度
- 记录玩家 HP、金币、关卡、星级、难度等信息

---

## 3. 技术栈

- Java 17
- JavaFX 21
- Maven
- JSON 本地存档

---

## 4. 运行要求

- JDK 17 或更高版本
- Maven 3.8+
- Windows 10/11

---

## 5. 启动方式

```bash
mvn javafx:run
```

---

## 6. 资源目录说明

- `src/main/resources/fxml/`：界面布局文件
- `src/main/resources/css/`：样式文件
- `src/main/resources/sprites/`：角色、怪物与背景素材
- `src/main/resources/data/`：题目、怪物、关卡配置数据

角色素材采用如下结构：

```text
src/main/resources/sprites/player/
src/main/resources/sprites/slime/
```

---

## 7. 存档位置

程序会在用户目录下生成存档文件：

```text
.quest-save.json
```

---

## 8. 当前实现说明

本项目采用“JavaFX + 本地 JSON”的单机架构，不依赖 Spring Boot 或 MySQL。后续如果需要扩展云存档、排行榜或成就系统，可以在当前架构基础上继续演进。
